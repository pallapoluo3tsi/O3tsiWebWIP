      Misys: Midas implementation

      The test solution for Misys: Midas leverages the FTS software components and frameworks to support any of your testing needs. This solution provides the test automation building blocks to support all of your requirements, use cases, test cases and test data. The solution reduces the upfront investment to build your test harnesses by leveraging frameworks, out-of-the-box automation and auto-generation techniques. The solution also eliminates the maintenance costs to support your test harness going forward. The test harness is built once and deployed across any version of the Midas platform.

      Misys: Midas Consulting & Implementation

      The consulting solution is focused on delivery of high-quality professional services around Misys: Midas implementations and upgrades, customization and integration, production support and adhoc projects.

      System Implementation & Upgrade

      Comprehensive end-to-end solution from project inception to delivery, including project planning, requirements analysis, design, development, test execution, component integration, functional, regression and performance/stress testing, defect reporting and remediation, user acceptance testing, deployment to Production, and post go-live support

      System Customization & Integration

      Integration with other third-party vendor platforms. Design and develop new interfaces into existing applications. Develop new front-end or back-end custom software solutions as needed by the client



      Production Support

      Daily operational and nightly batch support of system including monitoring, defect reporting, escalation and resolution

      Adhoc Projects

      On-demand implementation of small- to large-scale enhancements as requested by the client e.g. regulatory reporting changes

      Misys: Midas Out-of-the-box Coverage

      The software solution provides a multifaceted approach to test automation including front-end UI automation, end-to-end testing, data analysis, sophisticated reporting and automated documentation generation. Pre-built automation libraries comprising FTS scripts and datasources are also available out-of-the-box as part of the bundled package.

      Front-End UI Automation

      Comprehensive banking functionality including capture of deals, facilities, outstandings and report generation. Platform-agnostic support of Java, .NET, web-based and legacy applications thereby facilitating end-to-end testing of cross-technology interfaces around Misys: Midas


      Out-of-the-Box Libraries

      Pre-packaged scripts/datasources covering a wide range of assets (Deals, Facilities, Outstandings), lifecycle events (Commitment Increase, Share Adjustment, Portfoilio Transfer, Close etc ), business functionality (Fees, Principal & Interest, G/L, Reporting) and system testing (functional, regression, integration, performance, overnight batch)

      Data Analysis


      Sophisticated, bulk data comparison of all standard file formats (xml,csv,tab,xls.etc). Support of numeric tolerances (absolute and percentage), mapping files across application versions, ignore column list, audit of analysis and comments' entry



      Automated Reporting/Documentation

      Management-level reporting and presentation of test execution results, including dashboards, bar graphs/pie charts. Detailed log breakdown showing all PASS and FAIL scenarios. Historical reporting capability to track performance of regression test cycles. Automated audit- and compliance-ready Word document generated with test evidence such as real-time screenshots and embedded output extracts
